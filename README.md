# Sales Performance Dashboard

## Project Overview

This project analyzes sales data using SQL and Power BI to provide business insights.

## Tools Used

* SQL (MySQL Workbench)
* Power BI
* Excel

## Dashboard Features

* Total Revenue
* Total Profit
* Total Orders
* Average Sales
* Profit Margin %
* Region-wise Sales Analysis
* Profit by Category
* Top 10 Customers
* Top 10 Products
* Monthly Sales Trend
* Interactive Filters (Region & Category)

## Files Included

* Sales Performance Dashboard.pbix
* SQL_Queries.sql
* dashboard.png

## Key Insights

* West region generated the highest sales.
* Technology category contributed significant profit.
* Dashboard enables interactive filtering and analysis.
